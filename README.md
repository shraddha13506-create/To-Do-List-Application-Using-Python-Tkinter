# To-Do-List-Application-Using-Python-Tkinter
The To-Do List Application is a simple desktop application developed using Python and the Tkinter GUI library. It allows users to organize and manage their daily tasks through an easy-to-use graphical interface.

Users can enter a task in the input field and add it to the task list by clicking the "Add Task" button. Each task is automatically assigned a sequential number for better organization. The application also supports deleting selected tasks and automatically renumbers the remaining tasks to maintain a clean and ordered list.

The project demonstrates the fundamentals of GUI programming, event-driven programming, and Python data handling, making it an excellent beginner-level application for learning desktop software development.
